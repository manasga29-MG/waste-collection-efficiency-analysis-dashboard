# waste-collection-efficiency-analysis-dashboard
Interactive Power BI Dashboard analyzing waste collection efficiency using survey data with ward-wise performance, KPI analysis, and resident satisfaction insights.
Project Overview

This project analyzes waste collection efficiency using survey responses collected across multiple municipal wards. The dashboard provides insights into waste management practices, resident satisfaction, and operational performance to support data-driven decision-making.

Project Objectives
Analyze waste collection efficiency
Measure resident satisfaction
Compare ward-wise performance
Evaluate waste segregation practices
Identify operational gaps
Recommend improvements for municipal authorities
